Sådan virker det: En flersproget BERT-model (trænet til navnegenkendelse) kører direkte i din browser via ONNX Runtime + Transformers.js. Den finder personer, organisationer og steder i teksten. Modellen er ca. 180 MB og caches efter første indlæsning. Ingen data sendes til en server — alt foregår lokalt.
